## Challenges & Solutions

1. Learning New Tools Without Prior Experience
* Challenge: I had no prior experience using tools like Cursor, Claude Code, or GitHub workflows.
* Solution: I independently searched for tutorials, documentation, and examples online, and tested each tool step-by-step until I understood how they work.
  
2. Structuring a Portfolio Without Clear Instructions
* Challenge: There was no predefined format for the portfolio, so I needed to decide how to present my work clearly.
* Solution: I structured the project into clear sections (Overview, Tools, Steps, Approach, Sample Work) to ensure readability and logical flow.
  
3. Using AI Without Over-Reliance
* Challenge: AI tools can generate fast outputs, but not always accurate or high-quality.
* Solution: I used AI for drafting and ideation, but manually reviewed, edited, and validated all outputs to ensure quality and accuracy.

4. Writing Clear and Concise English Content
* Challenge: Ensuring the content is professional, concise, and easy to understand.
* Solution: I refined multiple drafts, simplified wording, and focused on clarity to make the content production-ready.

5. Working Independently Without Guidance
* Challenge: The project required self-direction without step-by-step instructions.
* Solution: Whenever I faced challenges, I actively searched for solutions (tutorials, documentation, AI tools) and continued progressing independently.
